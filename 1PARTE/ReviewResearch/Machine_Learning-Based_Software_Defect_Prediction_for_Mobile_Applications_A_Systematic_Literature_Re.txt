Software defect prediction studies aim to predict defect-prone components before the testing stage of software development process. The main benefit these models is that more resources can be allocated fault-prone modules effectively. While a few have been developed for mobile applications, systematic overview still missing. Therefore, we carried out Systematic Literature Review (SLR) study evaluate how machine learning has applied faults in applications.
